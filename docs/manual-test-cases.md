# 手动测试用例

## 首页

- 打开首页，看到城市、今日建议、上次洗车、保持天数、最佳日期、一周天气。
- 下拉刷新，首页重新请求天气建议。
- 点击“查看依据”，弹出洗车判断依据。
- 点击“我今天洗车了”，提示已记录，并把上次洗车更新为今天。

## 城市

- 进入城市页，看到常用城市列表。
- 未配置云环境时，输入“上”并搜索，只展示本地匹配城市。
- 配置云环境后，输入城市名并搜索，返回和风天气城市候选。
- 选择城市后返回首页，首页城市名更新。

## 记录

- 初次进入记录页，显示空状态。
- 点击记录后，显示当天记录。
- 点击清空，确认后记录清空。

## 云函数失败

- 未配置 `QWEATHER_API_KEY` 时，首页显示“云函数未配置 QWEATHER_API_KEY”。
- 输入无效城市 ID 时，云函数返回明确错误。
- 天气服务异常时，首页保留重试入口。
